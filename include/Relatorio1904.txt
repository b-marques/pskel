asn1parse: Faz o "parse" de uma sequência ASN.1(Formatação)


	-inform DER|PEM
        O formato de entrada.

       -in filename
           É determinado o arquivo de entrada.

       -out filename
       		Determina o arquivo de saída. Se nenhum arquivo de saída for especificado, então nenhum dado será gerado como saída.

       -noout
       		O output da versão parseada do arquivo de entrada não será mostrado.

       -offset number
       		A posição inicial do offset para começar o parse.

       -length number
       		O número de bytes para fazer o parse, por padrão é até o final do arquivo.

       -i
       		Faz a indentação do output de acordo com as estruturas.

       -oid filename
       		Um arquivo contendo OID (um identificador numérico para um conjunto de algoritmos criptográficos) adicionais.

       -dump
       		Escrever em um arquivo um dado desconhecido em formato hexadecimal.

       -dlimit num
       		Escreve em um arquivo um dado desconhecido em formato hexadecimal, contudo apenas os primeiros "num" bytes são usados como output.

       -strparse offset
       		Faz o parse dos octetos do objeto ASN.1 começando pelo offset fornecido. Essa opção pode ser usada para "entrar" numa estrutura aninhada.

       -genstr string, -genconf file
			Gerar um dado codificado baseado em uma string, arquivo ou os dois utilizando o formato ASN1_generate_nconf(função que gera uma codificação ASN1 de uma string).


ciphers: Converte uma lista de ciphers OpenSSL textual lista de algoritmos disponíveis para o processo de criptografia
